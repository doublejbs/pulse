# Pulse

React + MobX + TypeScript + Vite 프로젝트

## 시작하기

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

### 미리보기
```bash
npm run preview
```

## 프로젝트 구조

```
src/
  ├── stores/          # MobX stores
  │   ├── CounterStore.ts
  │   ├── RootStore.ts
  │   └── useStore.tsx
  ├── App.tsx         # 메인 앱 컴포넌트
  ├── main.tsx        # 엔트리 포인트
  └── index.css       # 글로벌 스타일
```

## 기술 스택

- React 18
- MobX 6
- TypeScript
- Vite
